
# ionic-capacitor-camera

It's a study Ionic project in order to learn how to capture photos from camera using Capacitor;

Running this app:

Install dependencies:

	$ npm install

Build and add Android or IOS platform:

	$ ionic build
    $ npx cap add android
    $ npx cap add ios

Run in debug mode:

	$ ionic serve
